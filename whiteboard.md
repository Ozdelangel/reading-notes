# Whiteboard Interview Advice

## Communicate
- as a lot of questions
- restate the question
- they should give you affirmation 
- ask about Test Cases
- write pseudocode and ask if it makes sense

It is always important to communicate and if you are absolutely stuck ask for help
You need to take their feedback and make sure you understand before you ever start coding
also often reference your visual it will be little bread crumbs if you get stuck
if you need to make adjustments in your code remember to go back to your algorithm and change it
Run through all the verification and make sure it makes sense
Above all take care of yourself. Remember to eat, sleep, and drink. getting lots of rests will help out a lot..
